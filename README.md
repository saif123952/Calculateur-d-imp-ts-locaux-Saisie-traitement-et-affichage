# Simulateur de Taxe Foncière - Android Lab

## Description
Cette application Android permet d'estimer rapidement le montant des impôts locaux pour une propriété. Elle a été développée dans le cadre d'un laboratoire de développement mobile.

![Capture d'écran de l'application](screenshot.png)

## Fonctionnalités
- Saisie de la surface habitable en m².
- Saisie du nombre de pièces.
- Option pour indiquer la présence d'une piscine.
- Calcul automatique basé sur les tarifs suivants :
    - **Base surface** : 2 DH / m²
    - **Majoration par pièce** : 50 DH
    - **Forfait piscine** : 100 DH

## Installation
1. Clonez ce dépôt.
2. Ouvrez le projet dans **Android Studio**.
3. Compilez et lancez l'application sur un émulateur ou un appareil physique.

## Technologies utilisées
- **Langage** : Java
- **UI** : XML (LinearLayout, Material Design)
- **Version minimum SDK** : 24+
